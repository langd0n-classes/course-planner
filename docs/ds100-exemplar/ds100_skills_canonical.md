# DS-100 Canonical Skills List

This list consolidates skills from historical trackers and aligns them to the
Spring 2026 learning modules.
It is the canonical reference for skill targeting in meeting plans.

- LM00-C02 Explain GenAI zones and integrity expectations.
- LM00-C03 Access and navigate Blackboard, Gradescope, Piazza, and TerrierGPT.

## LM-02 Data Claims

### Core skills

- LM02-C03 Create DataFrames from dictionaries and CSV files.
